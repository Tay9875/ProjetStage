Arrivé chez le disparu

Vous trouvez des signes d'effractions et des affaires éparpillés avec une note manuscrite que vous devez déchiffrer

Pour cela, vous devez répondre à 6 questions :

  • let : déclaration d'une variable à portée de bloc.

  • const : déclaration d'une constante.

  • function : déclaration d'une fonction.

  • if : structure de contrôle conditionnelle.

  • for : boucle qui répète jusqu'à ce qu'une condition spécifiée soit fausse.

  • return : arrêt de l'exécution d'une fonction et renvoi d'une valeur.

La lettre mentionne un nom et un lieu
