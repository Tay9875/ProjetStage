DETECTIVE CODE

LEARN TO CODE

Utiliser un mot de passe trouvé ailleurs ou résoudre un puzzle de substitution de lettres.

  • background-color: Permet de définir la couleur utilisée pour l'arrière-plan d'un élément.
  • font-size: Définit la taille de fonte utilisée pour le texte.
  • text-align: center: Définit l'alignement horizontal d'un élément de bloc ou de la boîte d'une cellule de tableau.
  • padding: L'espace entre le contenu et la bordure d'un élément.
  • margin: Définit la taille des marges sur les quatre côtés de l'élément.
  • color: Définit la couleur de premier plan d'un élément texte.
  • font-family: Permet de définir une liste, ordonnée par priorité.
  • display: none: Entraîner un retrait de l'arbre d'accessibilité.
