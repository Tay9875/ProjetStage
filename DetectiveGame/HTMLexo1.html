Hmmmmmm ! Peut-être qu'en étudiant ce document vous pourrez trouvé quelque chose d'intéressant.

En regardant le document, beaucoup de notions de texte basiques sont référencées. Analyser le code fourni afin de mieux comprendre l'utilisation des balises dans le body du code.

  • h: désigne un titre sur la page d'un site Interne.

  • p : représente un paragraphe de texte.

  • div : conteneur 'pur' pouvant être utiliser pour créer des blocs de contenu ou des mises en page.

  • a : un lien hypertexte permet de changer de page et/ou amener vers un endroit précis d'une page.

  • img : pour insérer une image en HTML.

  • ul,ol : liste d'éléments.

  • hr : permet de tracer un trait de séparation dans le texte.
