Regarder l’ordinateur du disparu en répondant à toutes les questions afin de débloquer le pc (cours sur les form).

Pour cela vous devez répondre à 6 questions.

  • form : représente un formulaire.

  • select : représente un contrôle qui fournit une liste d'options parmis lequelles l'utilisateur pourra choisir.

  • input : utilisé pour un contrôle interactif dans un formulaire web.

  • button : utilisé pour créer un boutton dans un formulaire.

  • textarea: utilisé pour créer une zone de texte multiligne où les utilisateurs peuvent saisir du texte.

  • label : permet de formuler un intitulé à un champ de formulaire.
